# Clinical Guidelines Placeholder

This file is intentionally lightweight and ready for future expansion.

Suggested content to add later:

- skin-condition-specific mask protocols
- contraindications
- severity-to-intensity mapping
- humidification usage rules
- thermal safety notes
- recommended session cadence

Current temporary guidance:

- Prefer conservative defaults when severity is high and the assessment confidence is uncertain.
- Use global temperature conservatively for sensitivity-related cases.
- Reserve higher brightness for cases that are not dominated by erythema or barrier sensitivity.
- Keep zone LED color recommendations editable by the operator.
