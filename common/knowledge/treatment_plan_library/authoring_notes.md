# Treatment Plan Library Authoring Notes

This directory is reserved for future treatment-planning knowledge.

Recommended usage:

- Add clinically reviewed mask-treatment cases to `case_memory.json`
- Add future hardware constraints to `mask_device_profile.json`
- Add professional guidance, contraindications, and protocol notes to `clinical_guidelines.md`

Important:

- Keep all structured codes in English
- Do not store API keys here
- Prefer concise, reusable guidance over long narrative notes
